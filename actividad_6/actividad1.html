En HTML5 no es necesario proporcinar valores para todos los stributos, como los controles, reproduccion automatica y atributos de bluque utlizados con el elemento <. Estos atributos son como interruptores de encendido/apagado. Si el atributo está presente, activa esa opción. Si se omite el atributo, la opcion está desactivida. Si el navegador no admite el elemento <video> o el formato de video utilizado, se mostrara lo que esté entre la <video>de apertura y las etiquetas de </video> de cierre.

preload: este atributo le indica al navegador qué hacer cuando se carga la pagina . puede tener uno de tres valores:
none: el navegador no debe cargar el video hasta que el usuario presiona Play.
auto: el navegador debe descargar el video cuando se carga la pagina.
metadata: el navegador debe simplemente recopilar informacion como el tamaño, el primer fotografo, la lista de pistas y la duracion.
