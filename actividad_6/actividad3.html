<audio>HTML5 introdujo el elemento <audio> para incluir archivos de audio es sus p√°ginas. Al igual que con el video HTML5, los navegadores esperan diferentes formatos para el audio. El elemento <audio> tiene un numero de atributos que le permiten controlar la reproduccion de audio:

src: este atributo especifica la ruta de acceso al archivo de audio

controls: este atributo indica si el reproductor debe mostrar los controles. Si no utiliza este atributo, no se mostran controles de manera predeterminada. Tambien puede especificar sus propios controles mediante JavaScript.

autoplay: La presencia de este atributo indica que el audio debe empezar a reproducirse automaticamente. (Se considera una mejor practica permitir que los visitantes elijan reproducir audio.)

loop: este atributo especifica que la pista de audio debe reproducirse de nuevo una vez que haya finalizado.
