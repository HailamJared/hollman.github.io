El elemento <div> permite agrupar un conjunto de elementos juntos en un cuadro de nivel de bloque. Por ejemplo, puede crear un elemento <div> para que contenga todos los elementos del encabezado del sitio (el logotipo y la navegacion en bloque)

En un navegador, el contenido del elemento <div> comenzara en una nueva linea, pero aparte de esto no hara ninguna diferencia en la presentacion de la pagina. sin embargo, el uso de un atributo ID o Class en el elemento <div>significa que puede crear reglas de estilo CSS para indicar cuanto espacio debe ocupar el elemento <div>en la pantalla y cambiar la aparencia de todos los elementos contenidos en el. Tambien pueden facilotar el seguimiento del codigo si ha utilizado elementos <div> para contener cada seccion de la pagina.

Dado que puede haber varios otros elementos dentro de un elemento <div> puede serutil agregar un comentario despues de la etiqueta </div> de cierre. Esto le permite ver claramente a que etiqueta de apetura se supone que correspone
